4B聊天室群规

最后修订于 主体113年11月2日

最后修订者 Mugunghwa

第一章 总则

  1. 第一条 为了保护4B聊天室群成员的合规权益，调解群成员纠纷，维护群聊整体秩序，制定本群规。
  2. 第二条 群内成员一律平等。
  3. 第三条 群内发生的民事行为依照本群规处理，群规未规定的，不予处理。
  4. 第四条 每位群成员进入群聊后即享有选举权和被选举权，被剥夺选举权利的群成员除外。
  5. 第五条 以下规则旨在维护群内良好的交流环境，望各位群成员共同遵守，营造和谐的聊天氛围。

第二章 违规处理

  1. 第六条 本群违规群成员的处理措施如下：
    ( 一 ) 移出群聊；
    ( 二 ) 剥夺选举权利；
    ( 三 ) 取消群组管理员身份；
  2. 第一节 移出群聊
    1. 第七条 移出群聊的期限为24小时以上144小时以下。特殊情况下，经群主批准可适当延长期限。
    2. 第八条 被移出群聊的群成员在期限内不得进入4B聊天室，期满后，该群成员可联系群主申请重新入群。
    3. 第九条 移出群聊的期限从执行之时起计算。
  3. 第二节 剥夺选举权利
    1. 第十条 剥夺选举权利是剥夺以下权利：
      ( 一 ) 选举权和被选举权；
      ( 二 ) 担任群主或群组管理员的权利；
    2. 第十一条 剥夺选举权利的期限为永久，仅在特赦日可恢复。
  4. 第三节 取消群组管理员身份
    1. 第十二条 此处理措施仅针对现任群组管理员。
    2. 第十三条 被取消群组管理员身份的群成员，六个月内不得再次担任群组管理员。

第三章 违规行为

  1. 第十四条 在群聊内发送连续三条及以上相同内容信息的或由群主主观判断有连续发送五条及以上无意义内容的，视为刷屏，首次违规将予以警告，再犯者将被移出群聊48小时。
  2. 第十五条 使用不当语言侮辱其他群成员或本群的，视为侮辱，将被移出群聊48小时，再犯者将被移出群聊144小时并剥夺选举权利。
  3. 第十六条 未经他人同意发布他人隐私信息（包括但不限于私密照片、住址、电话号码等）的，视为侵犯隐私，将被移出群聊72小时并剥夺选举权利。
  4. 第十七条 未经群主同意在群内发送别群的群号或入群二维码的，视为宣群，将被移出群聊24小时。
  5. 第十八条 对于群规未规定的违规行为，群主可在48小时内修改群规并进行处理。违规群成员在群规修改后的48小时内仍可被处罚。
  6. 第十九条 所有违规行为记录将记录于群公告中，以作警示。
  7. 第二十条 如发现违规行为，其他群成员可进行举报。举报需满足如下条件：
    ( 一 ) 仅接受截图作为证据；
    ( 二 ) 截图必须出现时间及日期；
    ( 三 ) 需在违规行为发生24小时内举报；
    ( 四 ) 举报需私信联系群主；

第四章 群组管理员

  1. 第二十一条 为促进群聊自由民主，协助群主管理群聊，设置群组管理员。
  2. 第二十二条 群组管理员拥有的权利有：
    ( 一 ) 发现群成员违规行为；
    ( 二 ) 执行除取消群组管理员身份以外的所有处理措施；
  3. 第二十三条 群组管理员按群主主观决定未能按规定执行处理措施两次及以上，视为滥用职权，将被取消群组管理员身份。
  4. 第二十四条 每年6月1日进行群组管理员选举，群主将通知每位群成员提名候选人（无被选举权者除外），并通过投票选出新群组管理员，每位群成员可投选两位或三位候选人。得票最多的前三名当选。如有平票，可能会进行第二轮投票。

第五章 特赦

  1. 第二十五条 每年2月16日和4月15日实施特赦，将清除所有违规行为记录（除滥用职权外）并恢复群成员的选举权利。被移出群聊的群成员可申请重新加入。

附则

  1. 第二十六条 若群规有错误或不足之处，请私信告知群主。
  2. 第二十七条 本群规最终解释权归群主所有。
  3. 第二十八条 本群规自主体113年10月4日起生效。
